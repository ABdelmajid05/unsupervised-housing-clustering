# Comparative Analysis of Unsupervised Clustering Algorithms on Housing Data

## Overview
This project presents a comparative analysis of multiple unsupervised clustering algorithms applied to a housing dataset. The study aims to explore hidden structures within the data and evaluate the performance of different clustering techniques using internal evaluation metrics and visual analysis.

## Implemented Algorithms
The following clustering algorithms were implemented and compared:

- K-Means
- DBSCAN
- Agglomerative Clustering
- Gaussian Mixture Models (GMM)

## Dataset
The dataset contains numerical housing-related features:

- MedInc
- HouseAge
- AveRooms
- AveBedrms
- Population
- AveOccup
